## Description

Вариант блога демонстрирующий список постов.

### Функционал:

  • Создание нового поста с вводом заголовка
  • Удаление поста
  • Редактирование заголовка поста


## Getting Started

### Dependencies

 • npm > 10.2.4
 • vite 
 • pinia
 • unix

### Installing

```npm install```

### Executing program

```npm run dev```

### Linter

```npm run lint```
